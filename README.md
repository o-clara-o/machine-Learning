# 图像识别项目：利用机器学习方法研究气泡筏中的动力学行为（院优秀毕业论文）

## 项目简介
基于卷积神经网络的图像分类系统，实现对气泡筏中气泡的识别，并生成简单的数字孪生图像。

## 技术栈
- Python 3.8
- YOLOv8
- PyTorch
- OpenCV

## 核心功能
- 图像预处理（缩放、归一化）
- 模型训练
- 单张图片预测和批量测试
